## substrate-p2p

Substrate based chains p2p network utilities.
This project is a fork of https://github.com/lexnv/subp2p-explorer but customised to my needs

## Discovery

Crawls the p2p network. For more details see
the [kad-dht spec](https://github.com/libp2p/specs/blob/master/kad-dht/README.md).

## Notifications

Ability to stream notifications for a given protocol.
